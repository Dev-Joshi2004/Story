code file_name ====> is use to open a file 
jis directory mein ham kaam karte hai usse working directory bolte hai, agr hame apne code ka track rakhna hai to hame us working directory ko "Staging Area" mein save karna hoga.
us file ko staging area mein add karne ke liye ham "git add file_name" command ka use karte hai.
kisi file ko commit karne ke liye ham "git commit -m `commit message`" -m flag is used to add some messages in your commit taki aapko pata chalta rahe ki apne last commit mein kya kara tha.
